- **버전 관리**: Git/GitHub
- **개발 환경**: 로컬 + 웹 기반

## 핵심 기능
1. **자동 파일 생성**: 학생별 상담카드 파일 자동 생성
2. **양방향 동기화**: 교사 ↔ 학생 데이터 실시간 동기화
3. **스마트 업데이트**: 변경된 파일만 선별적 동기화
4. **자동 정렬**: 학생 번호순 시트 정렬
5. **트리거 관리**: 자동화 설정 ON/OFF
